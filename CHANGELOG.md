# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-11-07

### Added
- Initial Apache Superset configuration setup
- Main configuration file (`superset_config.py`) with comprehensive settings
- Docker Compose setup with PostgreSQL and Redis
- Environment configuration template (`.env.example`)
- Python requirements file with Superset and database drivers
- Initialization script for database setup
- Comprehensive README with setup and usage instructions
- `.gitignore` and `.dockerignore` files
- Support for multiple languages including Polish

### Features
- Pre-configured security settings
- Redis caching integration
- PostgreSQL database support
- Email/SMTP configuration for alerts
- CORS configuration
- Feature flags for modern Superset features
- Docker-based deployment
- Healthchecks for all services

### Configuration
- Application main context setup
- Database connection configuration
- Cache configuration
- Authentication and security settings
- Email server configuration
- Localization support
